Day 1 Summary — AI-Procure (Case 2)

1. Цель проекта
Создать ИИ-агента, который автоматически анализирует тендеры: извлекает параметры, ищет риски, подбирает поставщиков и формирует отчёт.

2. Параметры, которые извлекает агент
- Предмет закупки
- Цена
- Сроки / дедлайны
- Требования
- Заказчик
- Условия оплаты
- Гарантии и штрафы

3. Риски, которые определяет агент
- Завышенная цена
- Аффилированность (совпадение ИИН/БИН, адресов)
- Нереалистичные сроки
- Скрытые условия / штрафы

4. Источники данных
- Госзакупки РК
- Kaggle / OpenContracting
- Синтетические тендеры (PDF)
